## E-Commerce Sales Dashboard & Business Analysis (Excel)

##  Project Overview

This project focuses on analyzing E-Commerce sales data and building an interactive dashboard using Microsoft Excel.
The goal is to extract meaningful business insights and track key performance metrics such as revenue, profit, and customer performance.

---

##  Objectives

* Analyze overall sales performance
* Identify top customers and revenue contributors
* Evaluate profit across product categories
* Understand monthly sales trends
* Support data-driven decision making

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Excel Formulas (SUM, IF, COUNTA, etc.)

---

## Key Features

### 🔹 KPI Metrics

* Total Revenue
* Total Orders
* Total Profit
* Average Order Value (AOV)

### 🔹 Analysis Performed

* Top 10 Customers by Sales
* Country-wise Sales Analysis
* Category-wise Profit Analysis
* Monthly Sales Trend Analysis

---

##  Dashboard

The dashboard provides a clear and visual representation of sales performance:

* KPI cards for quick insights
* Line chart for monthly trends
* Bar charts for customer and country analysis
* Column chart for category-wise profit

---

##  Key Insights

* A small group of customers contributes significantly to total revenue
* Certain countries generate higher sales compared to others
* Some product categories show lower or negative profit margins
* Sales trends vary across different months, indicating seasonal patterns

---

##  Project Structure

* 'Excel Project.xlsx` → Main Excel dashboard file

---

##  How to Use

1. Download the Excel file
2. Open in Microsoft Excel
3. Explore the dashboard and charts
4. Review insights from different analyses
